Suggestion Categories (Completing stylistic fragments turned "on")

The Grammarly Text Editor Plugin on this page is set to show suggestions for the "stylisticFragments" writing suggestion category. This category, which is muted by default, suggests completing all incomplete sentences, including stylistic sentence fragments that may be intentional.
